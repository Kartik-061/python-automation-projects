# Python Automation Projects

A collection of Python automation scripts built using Selenium WebDriver 
that mimic human behavior to automate repetitive web tasks.

## Projects

### 1. JumpTask Automation Bot
Automates task completion on the JumpTask platform.

**Features:**
- Reads task descriptions and finds matching YouTube videos
- Implements human-mimic delays and randomized mouse behavior
- Tiered link priority system for intelligent task detection
- Fingerprint memory to avoid repeating completed tasks

### 2. Microsoft Rewards Bot
Automates Bing searches to earn Microsoft Reward points daily.

**Features:**
- Human-like typing simulation (character by character)
- Edge browser user-agent spoofing for anti-detection
- Indian search queries for natural behavior
- Scheduled search automation

## Technologies Used
- Python
- Selenium WebDriver
- WebDriver Manager
- Chrome/Edge Browser Automation

## Setup
1. Install dependencies:
   pip install selenium webdriver-manager
2. Update your credentials in the script
3. Run the script:
   python auto2.py
